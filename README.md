# PoGoBag

**Analyze** and **Share** your PokemonGo Inventory online!

Feel free to contribute and make pull request.

To Setup:
```
git clone https://github.com/dphuang2/PoGoBag.git
cd PoGoBag
rake db:setup
rails s
navigate to localhost:3000 in browser
```
